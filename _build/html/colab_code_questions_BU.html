Question#

Print the mean, median, mode and standard deviation of the follwoing list of nubers: \([12, 3, 12, 9, 7, 3, 25,6]\) using the statistics module.

  • You can use dir(statistics) or help(statistics) for the list of available functions in statistics.

Solution

import statistics

print(statistics.mean([12, 3, 12, 9, 7, 3, 25,6]))
print(statistics.median([12, 3, 12, 9, 7, 3, 25,6]))
print(statistics.mode([12, 3, 12, 9, 7, 3, 25,6]))
print(statistics.pstdev([12, 3, 12, 9, 7, 3, 25,6]))    # population std
print(statistics.stdev([12, 3, 12, 9, 7, 3, 25,6]))     # sample std
